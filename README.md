## Feature

- support deploy to github
- markdown source
- RSS
- friendly to SEO
- support local preview

## Usage

### Install

```bash
go get -v github.com/biezhi/bar
```

Or

[Download]() this binary package.

### Write article

```bash
bar init blog
cd blog
bar s
```
